Material Line Icons

This icon set has 24x24 icons designed to Material Design guidelines that are used for 'Round' and 'TwoTone' icons in official Material Design Icons with one minor change: thin 1px stroke is allowed where it makes sense.

Icons use stroke that can be animated on first render. All you have to do is add class "iconify--line-md" to SVG element (Iconify SVG framework does it automatically) and include stylesheet from https://code.iconify.design/css/line-md.css

To refresh animation you can either re-render icon or use display to hide and show it.
